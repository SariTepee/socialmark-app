# socialmark-app

## Launch Project

### Step 1 - Project setup

```
npm install
or
yarn install
```

### Step 2 - Run Frontent (Terminal 1)

```
npm run serve
or
yarn serve
```

### Step 3 - Listen Fake Api (Terminal 2)

```
json-server --watch db.json
```

### Step 4 - Run Server (Terminal 3)

```
cd server
and
npm run start
or
yarn start
```
